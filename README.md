# Física Aplicada a la Tecnología de Alimentos

Sitio estático para publicar en GitHub Pages y enlazar desde Moodle.

Estructura:

- `index.html`: programa general de la materia.
- `unidad1/` a `unidad8/`: carpetas de unidades.
- `unidad2/`: contiene el apunte interactivo de Cinemática con video e imagen.

Para publicar en GitHub Pages: subir todo el contenido del directorio a un repositorio público y activar Pages desde `main` / root.
